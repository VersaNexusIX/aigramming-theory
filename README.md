# Aigramming

**Aigramming** is a next-generation programming paradigm in which program instructions are written as natural language prompts, interpreted directly by an artificial intelligence model embedded within the device.

This concept represents the logical continuation of the programming language evolution that has unfolded over several decades, from binary machine code to high-level interpreted languages.

## Conceptual Overview

Aigramming introduces a new abstraction layer atop existing paradigms. A program file within the Aigramming ecosystem carries a file extension that reflects the AI model serving as its interpreter, such as `.gpt`, `.gemini`, or `.llama`. The contents of such a file consist of plain readable text that can be viewed using standard utilities such as `cat`; however, execution is handled entirely by an AI runtime operating locally on the device.
